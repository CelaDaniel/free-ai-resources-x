# Recommender Systems

- [Google: Recommendation Systems Course](https://developers.google.com/machine-learning/recommendation) - Free comprehensive guide to building recommendation systems.
- [Microsoft Recommenders Repository](https://github.com/microsoft/recommenders) - Free best practices and reference implementations for recommender systems.
- [Coursera: Recommender Systems Specialization by Minnesota](https://www.coursera.org/specializations/recommender-systems) - Free to audit, in-depth course on recommendation algorithms.
- [Fast.ai: Collaborative Filtering](https://course.fast.ai/) - Free practical deep learning with collaborative filtering examples.
- [Surprise Library Documentation](http://surpriselib.com/) - Free Python library for building and analyzing recommender systems.
- [RecSys Academy](https://github.com/ACMRecSys) - Free ACM RecSys conference resources and educational materials.
