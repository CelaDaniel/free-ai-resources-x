# Graph Neural Networks (GNNs)

- [Stanford CS224W: Machine Learning with Graphs](http://web.stanford.edu/class/cs224w/) - Free comprehensive course on graph neural networks and graph ML.
- [PyTorch Geometric Tutorials](https://pytorch-geometric.readthedocs.io/en/latest/tutorial/index.html) - Free tutorials for the leading GNN library.
- [Deep Graph Library (DGL) Tutorials](https://docs.dgl.ai/tutorials/blitz/index.html) - Free comprehensive tutorials on building GNNs with DGL.
- [Petar Veličković: GNN Course](https://www.youtube.com/watch?v=uF53xsT7mjc) - Free video course by Cambridge researcher on GNN theory and practice.
- [Graph Representation Learning Book](https://www.cs.mcgill.ca/~wlh/grl_book/) - Free online book on graph neural networks.
- [NetworkX Documentation](https://networkx.org/documentation/stable/tutorial.html) - Free tutorials on graph analysis with Python.
