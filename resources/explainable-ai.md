# Explainable AI (XAI)

- [Christoph Molnar: Interpretable Machine Learning](https://christophm.github.io/interpretable-ml-book/) - Free comprehensive online book on model interpretability.
- [SHAP Documentation & Tutorials](https://shap.readthedocs.io/en/latest/) - Free guides for the leading explainability framework.
- [Google: Explainable AI Resources](https://cloud.google.com/explainable-ai) - Free tools and documentation for interpretable AI.
- [Coursera: Interpretable Machine Learning](https://www.coursera.org/learn/interpretable-machine-learning) - Free to audit, course on XAI techniques.
- [LIME Documentation](https://lime-ml.readthedocs.io/en/latest/) - Free tutorials for Local Interpretable Model-agnostic Explanations.
- [Papers: XAI Research](https://arxiv.org/list/cs.AI/recent?keywords=explainable) - Free access to latest explainability research.
