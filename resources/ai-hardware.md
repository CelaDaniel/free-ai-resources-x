# AI Hardware & Infrastructure

- [NVIDIA Deep Learning Institute: Free Courses](https://www.nvidia.com/en-us/training/) - Free courses on GPU computing and AI acceleration.
- [Google Cloud: TPU Tutorials](https://cloud.google.com/tpu/docs/tutorials) - Free guides on using TPUs for ML training.
- [AWS: EC2 for Machine Learning](https://aws.amazon.com/getting-started/hands-on/get-started-dlami/) - Free tier tutorials on cloud ML infrastructure.
- [Distributed Training with Horovod](https://horovod.readthedocs.io/en/stable/) - Free documentation on distributed deep learning.
- [Ray.io: Distributed Computing](https://docs.ray.io/en/latest/ray-overview/getting-started.html) - Free tutorials on scaling ML workloads.
- [Edge AI and TinyML](https://www.edx.org/learn/computer-science/harvard-university-cs249r-tiny-machine-learning-tinyml) - Free edX course on ML for edge devices.
