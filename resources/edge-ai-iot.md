# Edge AI & IoT

- [Harvard: TinyML Course](https://www.edx.org/learn/computer-science/harvard-university-cs249r-tiny-machine-learning-tinyml) - Free course on machine learning for embedded systems.
- [Edge Impulse: ML on Edge Devices](https://www.edx.org/learn/embedded-systems/edge-impulse-introduction-to-embedded-machine-learning) - Free course on deploying ML to edge devices.
- [TensorFlow Lite Tutorials](https://www.tensorflow.org/lite/guide) - Free guides for running ML models on mobile and embedded devices.
- [Arduino: Machine Learning](https://docs.arduino.cc/tutorials/nano-33-ble-sense/get-started-with-machine-learning/) - Free tutorials on ML for Arduino microcontrollers.
- [NVIDIA Jetson AI Courses](https://developer.nvidia.com/embedded/learn/jetson-ai-certification-programs) - Free AI certification for edge computing.
- [Google Coral: Edge ML Resources](https://coral.ai/docs/) - Free documentation for edge TPU development.
